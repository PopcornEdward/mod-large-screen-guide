# off-site-selector

## 组件介绍

【异地查询】数据关联组件

## 使用场景

实名制项目存在本地和异地数据，这个组件用于切换整块大屏的数据

## Attributes

| 参数  | 说明                        | 类型   | 可选值 | 默认值 |
| ----- | --------------------------- | ------ | ------ | ------ |
| label | checkbox对应label的业务名称 | String |        | `""`   |




## $data

| 参数    | 说明         | 类型    | 可选值 | 默认值 |
| ------- | ------------ | ------- | ------ | ------ |
| checked | 按钮选中状态 | Boolean |        | true   |

## methods

| 名称      | 描述                 |
| --------- | -------------------- |
| fn_change | 同步改变全局异地变量 |
